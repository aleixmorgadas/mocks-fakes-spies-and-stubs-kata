# Mocks, Fakes, Spies and Stubs Kata

## Requirements

- Java 8
- Gradle
- An IDE, I would suggest IntelliJ

## Setup Environment

1. Clone the repository

    `git clone git@github.com:aleixmp/mocks-fakes-spies-and-stubs-kata.git`
2. Check all tests are green
    `./gradlew test`
